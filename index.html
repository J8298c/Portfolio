Projects
Apollo

Apollo is a workout logging application, which I created to log my own workouts as I found most of the workout applications available, to be very cubersome when trying to workout. Apollo lets users add and edit workouts stored in the database. Most of the workouts in were inspired by the folks over in Super Hero Jacked.

Github Card

Github Card lets users search github users. Users can see and easily access their repos, followers and other useful information. Github Card was initially developed while I was tinkering around with Github api.
