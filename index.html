Github_card

Github card uses the github api to retrieve users information and display it in a card based profile. Built with React and Redux to handle the new javascript fetch api to retrieve users information from the github api. Testing is done using Airbnb's Enzyme and Chai, React-Router handles the applications routing. Work is still being done on this Repo to include adding user authentication in order to let users sign in and track issues with thier repos.


Technology Used: React | javaScript | Redux | Webpack | Enzyme | React-Router | Chai | Yarn
Project Link

Yumshub

Yumshub is a clone of grubhub built with React, React-Router and Redux and uses firebase as its database. Yumshub display a list of resturants and allows you to search by the the type of food i.e.(American, Spanish). Once inside the restuarant page the user can see the restaurants menu, and add menu items to thier shopping cart. Yumshub was designed using React-Bootstrap alongside custom css. Tetsing is done using Airbnb's Enzyme, Chai alongside Sinon.


Technology Used: HTML5 | javaScript | CSS3 | React | React-router | Redux | React-Bootstrap | Webpack | Enzyme | Sinon | Chai | Firebase | Yarn
